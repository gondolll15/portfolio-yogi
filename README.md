 # Portfolio Website - Yogi Pratama

# Deskripsi
Website ini merupakan portfolio pribadi yang dibuat untuk menampilkan informasi tentang diri saya, skill yang dimiliki, serta project yang pernah saya kerjakan.
Website ini dirancang dengan tampilan modern, interaktif, dan responsive sehingga dapat diakses dengan baik di berbagai perangkat seperti laptop maupun smartphone.

---

# Tujuan
* Menampilkan profil dan identitas diri secara online
* Menunjukkan kemampuan dalam bidang Web Development
* Menjadi media dokumentasi project yang telah dibuat
* Melatih kemampuan dalam HTML, CSS, dan JavaScript

---

# Fitur Utama

* Tampilan modern dengan animasi menarik
* Efek typing text (Typed.js)
* Animasi Lottie (ilustrasi bergerak)
* Navigasi halaman (Home, Tentang, Skill, Project, Contact)
* Responsive design (mobile & desktop friendly)
* Animasi scroll (reveal effect)

---

# Teknologi yang Digunakan

* HTML → Struktur halaman website
* CSS → Desain dan tampilan visual
* JavaScript → Interaksi dan animasi
* Typed.js → Efek teks berjalan
* Lottie Player → Animasi ilustrasi
* Font Awesome → Icon sosial media

---

# Struktur Halaman

* Home → Tampilan utama (hero section)
* Tentang → Informasi tentang diri
* Skill → Kemampuan programming & tools
* Project → Daftar project yang pernah dibuat
* Contact → Sosial media dan kontak

# Cara Menjalankan Project

1. Clone repository ini:

   ```
   git clone https://github.com/gondolll15/portfolio-yogi.git
   ```
2. Masuk ke folder project:

   ```
   cd nama-repository
   ```
3. Buka file `index.html` di browser
4. Website akan langsung tampil
